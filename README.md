# AI脑暴 - 引导式AI创意讨论平台

> **v1.0.0** | MIT License

AI脑暴是一个引导式AI创意讨论平台。通过AI主持人引导对话、多位虚拟专家从不同专业角度参与讨论，帮助你高效梳理创意，并自动生成会议纪要和专业文档（PRD/SPEC/用户故事地图/技术方案/市场分析报告/行动计划）。

## 功能特性

### 核心引擎

- **多Agent编排**：AI主持人引导讨论并指定1~3位专家发言，5位预设专家（产品经理、技术架构师、市场分析师、UX设计师、增长黑客）从各自专业角度提供深度分析
- **发散-收敛双阶段**：讨论显式划分为发散（探索可能性）与收敛（评估筛选、推动共识）两阶段，主持人与专家提示词动态调整
- **专家建设性辩论**：专家被引导主动寻找分歧、敢于反驳并提出替代方案，推动逼近最优方案而非虚假共识
- **中场暂停与继续**：每5轮自动暂停生成中场总结，用户可补充信息后继续剩余轮次，暂停点持久化可恢复
- **阶段总结**：每4轮自动生成讨论总结，也可手动触发，梳理已讨论要点和待解决问题

### 知识增强

- **RAG 检索增强**：知识条目自动生成 embedding 向量，讨论时语义检索 top-K 相关知识注入上下文；支持独立 Embedding 端点配置，不兼容时自动降级为全量检索
- **专家长期记忆**：自动提取专家洞察、偏好、经验教训三类长期记忆，跨项目积累，发言时语义检索注入相关记忆
- **共享知识库**：每轮讨论后异步提取结构化知识条目（事实/决策/共识/分歧/待解问题），去重并按优先级注入后续上下文
- **智能推荐专家组合**：新建项目时一键推荐最适合的 2-4 位专家，LLM 分析推荐理由
- **动态总结**：共识或分歧知识新增 ≥2 条时自动触发，无需等待固定间隔

### 讨论体验

- **流式对话**：基于 SSE 的实时流式输出，打字机效果，角色分色显示
- **用户干预指令**：输入 `/focus`、`/ask`、`/redirect` 主动干预讨论方向
- **动态专家管理**：讨论过程中可邀请新专家或移除已有专家（前3轮允许变更）
- **消息编辑**：编辑已发送消息，系统自动删除后续对话并重新生成
- **文件附件**：上传文本类文件（.txt/.md/.csv/.json/.log），解析后注入上下文
- **软停止**：让当前专家说完后自然结束，不中断流式输出
- **空响应重试**：LLM 返回空响应时自动重试 1 次

### 可视化

- **讨论回放**：消息级撤销/重做，回放播放器支持速度控制和进度拖拽
- **4种可视化视图**：时间线、词云、争议矩阵、热度图，所有组件懒加载
- **专家画像仪表盘**：跨项目统计专家参与数据、共识贡献率、高频话题
- **思维导图**：基于纪要流式生成 Markdown 思维导图，标注共识与分歧，支持导出 SVG/PNG
- **状态看板**：实时反映讨论阶段、专家轮次进度、共识分歧计数

### 成果输出

- **会议纪要**：自动生成结构化纪要（讨论主题、核心观点、主要分歧、下一步建议）
- **6种文档草稿**：PRD、SPEC、用户故事地图、技术方案、市场分析报告、行动计划
- **讨论历史导出**：支持 JSON / Markdown / TXT 三种格式

### 基础设施

- **中英文双语**：基于 next-intl，通过 Cookie 检测语言偏好
- **Ollama 离线模式**：支持本地 Ollama 作为 LLM Provider，完全离线使用
- **Electron 桌面应用**：支持打包为 Windows 桌面应用
- **状态快照**：每 20 条消息自动创建快照，优化长讨论性能

## 技术栈

| 分类 | 技术 |
|------|------|
| 框架 | Next.js 16（App Router + Turbopack） |
| 语言 | TypeScript |
| 数据库 | SQLite + Prisma 7 |
| AI | Vercel AI SDK v6 + @ai-sdk/openai v3 |
| 样式 | Tailwind CSS v4 |
| 状态机 | XState v5 |
| 通信 | SSE（Server-Sent Events） |
| 桌面 | Electron 42 + electron-builder |

> **注意**：本项目使用 `openai.chat()` 强制走 Chat Completions API，确保兼容阿里云百炼、小米MiMo等国内 API。

## 安装

### 方式一：桌面应用（推荐）

1. 下载 `AI脑暴-1.0.0-x64.exe` 安装包
2. 双击运行，选择安装目录
3. 安装完成后从开始菜单或桌面快捷方式启动

### 方式二：本地运行

1. 安装 [Node.js 20+](https://nodejs.org/)（推荐 22 LTS）
2. 下载项目代码，在项目目录下执行：

```bash
npm run setup
npm run dev
```

3. 打开浏览器访问 `http://localhost:3000`

> Windows 用户可双击 `start.bat` 使用一键启动菜单。

## LLM 配置

支持任意 OpenAI 兼容格式的 LLM API。有两种配置方式：

### 方式A：页面配置（推荐）

启动后点击右上角「设置」，填写 API 地址、API Key 和模型名称。配置存储在浏览器本地。

### 方式B：.env 文件

在项目根目录创建 `.env` 文件：

```env
LLM_BASE_URL="API地址"
LLM_API_KEY="你的API Key"
LLM_MODEL="模型名称"
```

**常见 API 配置示例**：

| Provider | Base URL | 模型示例 |
|----------|----------|---------|
| 阿里云百炼 | `https://dashscope.aliyuncs.com/compatible-mode/v1` | `deepseek-v4-pro` |
| 小米 MiMo | `https://api.xiaomimimo.com/v1` | `mimo-v2.5-pro` |
| Ollama 本地 | `http://localhost:11434/v1` | `qwen2.5:14b` |

## 使用流程

### 1. 创建项目

在首页点击「新建脑暴」，输入讨论主题，选择至少2位专家参与。也可使用内置模板（新产品立项、技术选型、市场分析、产品复盘）。

### 2. 开始讨论

在输入框中输入你的想法，AI主持人会概括输入并指定最合适的专家回应。专家以 [HOOK] 问句结尾引导你进一步思考。

### 3. 互动深入

- 回应专家问句，推动讨论深入
- 输入 `/` 使用干预指令（`/focus`、`/ask`、`/redirect`）
- 点击「+」管理参与专家（前3轮可变更）
- 上传文本附件供专家引用
- 编辑已发送消息重新生成对话
- 点击「进入收敛」切换到收敛阶段，推动达成共识
- 可随时点击「软停止」让当前专家说完后自然结束

### 4. 结束并生成纪要

点击「结束脑暴并生成纪要」，系统自动生成结构化会议纪要。

### 5. 生成文档与思维导图

在成果页可：
- 选择文档类型生成草稿（PRD/SPEC/用户故事地图/技术方案/市场分析报告/行动计划）
- 生成思维导图并导出 SVG/PNG
- 导出讨论历史（JSON/Markdown/TXT）

## 预设专家角色

| 专家 | 关注领域 |
|------|----------|
| 产品经理 | 用户价值、需求优先级、MVP范围、产品定位 |
| 技术架构师 | 技术可行性、架构选型、性能瓶颈、开发成本 |
| 市场分析师 | 市场规模、竞争格局、差异化策略、获客渠道 |
| UX设计师 | 用户旅程、信息架构、交互效率、情感化设计 |
| 增长黑客 | 获客漏斗、病毒系数、留存曲线、变现模式 |

> 支持在「专家管理」页面创建自定义角色，设置名称、人设、关注领域和配色。

## 开发

```bash

npm run build        # 生产构建
npm run electron:build # Electron 打包（产物：AI脑暴-1.0.0-x64.exe）
```

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 连接测试失败 | 确认 API 地址/Key/模型名称正确，使用 Chat Completions 兼容端点 |
| AI 回复空白 | 检查模型名称，使用「测试连接」验证（空响应会自动重试1次） |
| NODE_MODULE_VERSION 不匹配 | 执行 `npm run rebuild:sqlite` 重新编译原生模块 |
| 端口被占用 | 执行 `npm run kill-port` 清理 3000 端口 |
